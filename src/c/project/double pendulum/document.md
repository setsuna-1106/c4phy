## 1. 题目描述
模拟一个受重力作用的双摆系统。给定初始角度，计算两个摆锤随时间演化的轨迹。通过能量守恒检验数值精度。

## 2. 物理模型
使用牛顿力学来解决该问题

**自由度**：两杆与竖直方向的夹角，$\phi_1$,$\phi_2$

**控制方程**：利用牛顿第二定理和转动的相关方程，可以推导出任意给定$\phi_1$,$\phi_2$ 的角加速度

**参数表**：
| 参数 | 符号 | 值 | 单位 |
|------|------|----|------|
| 摆长 1 | $L_1$ | 300 | cm |
| 摆长 2 | $L_2$ | 200 | cm |
| 质量 1 | $m_1$ | 1 | kg |
| 质量 2 | $m_2$ | 1 | kg |
| 重力加速度 | g | 980 | cm/s² |

**初始条件**
利用随机数设置两杆与竖直方向的夹角$\phi_1$ $\phi_2$
设置角速度和角加速度为0

**假设/近似**
无空气阻尼
采用刚体模型，假设两杆为轻杆(质量为零)

## 3. 数值方法
利用半隐式Euler方法进行数值模拟
加速度使用当前值，一阶近似得到速度，再由新的速度值得到新的角度值


## 4. 实现细节
将初始化，步进函数，绘制双杠写做独立的函数，由main组织并绘图

### 文件列表

| 文件 | 职责 |
|------|------|
|`main.c`|主函数入口，数值模拟以及可视化|


### 核心函数
| 函数名                   | 功能描述                         |
|--------------------------|----------------------------------|
| `init_slover()`          | 初始化求解器                     |
| `step()`                 | 利用Euler步进                    |
| `draw_double_pendulum()` | 绘制双杠，以及下端小球的运动轨迹 |
| `main()`                 | 主函数入口，绘图                 |


### 输入和输出
无需输入，所有参数都通过init（）函数初始化得到
无输出，运行的结果直接通过raylib 可视化呈现

## 5. 编译 & 运行

### 依赖
- GCC / Clang
- Raylib

### 编译
```bash
# Raylib 项目
gcc main.c -o main $(pkg-config --cflags --libs raylib) -lm
```

### 运行
```bash
./main
```

## 6. 验证 & 结果

对模拟结果进行守恒量检验：
记录初始时刻的能量，在每次调用step()函数时，同时计算此时的能量
从而计算出相对能量误差
可以看到相对能量误差保持在1%左右，可以视为较为精确的模拟
