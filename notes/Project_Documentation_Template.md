# 计算物理项目文档通用模板

> 适用于本仓库所有 `src/c/project/` 和 `src/python/project/` 下的小型项目。

---

## 模板结构

```
# 项目名称
项目简介 (~200字).

---

## 目录
1.  [题目描述]
2.  [物理模型]
3.  [数值方法]
4.  [实现细节 / 代码结构]
5.  [编译 & 运行]
6.  [验证 & 结果]
7.  [参考资料]
```

---

## 1. 题目描述

**任务**: 这个项目要算什么 / 模拟什么？

**写法**: 现在时陈述，第三人称。不需要公式。

**示例**:
> 模拟一个受重力作用的双摆系统。给定初始角度，计算两个摆锤随时间演化的轨迹。通过能量守恒检验数值精度。

---

## 2. 物理模型

描述系统的物理学——控制方程、变量、假设、参数。

### 自由度
列出描述系统的变量（如 theta1, theta2）。

### 控制方程
用 LaTeX 写出 ODE / PDE。

### 参数表

| 参数 | 符号 | 值 | 单位 |
|------|------|----|------|
| 摆长 1 | L1 | 300 | cm |
| 质量 1 | m1 | 1 | (无量纲) |
| 重力加速度 | g | 980 | cm/s^2 |

### 初始条件
列出所有变量的初始值及物理含义。

### 假设 / 近似
- 无阻尼 / 有阻尼？
- 质点模型 / 刚体？
- 小角度近似？
- 其他简化...

### 守恒量
系统存在什么守恒量？总能量 / 动量 / 角动量是否应为常数？

---

## 3. 数值方法

怎么用计算机逼近连续方程。

### 算法
- **算法名**: Euler / RK4 / Verlet / Leapfrog / ...
- **离散格式**: 用伪代码或公式写出一时间步递推

```
# 例：半隐式 Euler
alpha(t) = 加速度(theta(t), omega(t))
omega(t + dt) = omega(t) + alpha(t) * dt
theta(t + dt) = theta(t) + omega(t + dt) * dt


### 时间步长
dt 的值及选取依据。

### 终止条件
运行多久停止？固定步数 / 固定物理时间 / 手动关闭？

### 已知局限
这个方法在这里有什么问题？能量漂移？刚度问题？发散风险？

---

## 4. 实现细节 / 代码结构

描述代码是怎么组织的（别只贴源码）。

### 文件列表

| 文件 | 职责 |
|------|------|
| `main.c` | 主程序入口，初始化 + 主循环 |
| `solver.c` | 数值积分器 |
| `visual.py` | 读取输出 CSV 并绘图 |

### 核心函数

| 函数 | 签名 | 职责 |
|------|------|------|
| `step()` | `void step(double dt)` | 推进一个时间步 |
| `init_solver()` | `void init_solver()` | 初始化物理量 |

### 输入 / 输出
- 读什么？（配置文件 / 命令行参数 / 硬编码常量）
- 写什么？（CSV / 二进制 / 直接在屏幕上渲染？）

### 关键常量

| 宏 / 常量 | 值 | 含义 |
|-----------|----|------|
| `FPS` | 500 | 渲染帧率 |
| `SUBSTEPS` | 1000 | 每帧的子步数 |

## 5. 编译 & 运行

### 依赖
- GCC / Clang
- Raylib 5.x (如涉及)
- Python 3.x + numpy + matplotlib (如涉及)

### 编译

```bash
# Raylib 项目
gcc main.c -o main $(pkg-config --cflags --libs raylib) -lm

# 纯数值 C 项目
gcc -O3 main.c -o main -lm
```

### 运行

```bash
./main
```

## 6. 验证 & 结果

你怎么知道结果是对的？**至少选一项**：

- **守恒量检验**: 能量 / 动量 / 角动量是否守恒？给出漂移率
- **收敛性测试**: dt 减半时误差如何变化？
- **解析解对比**: 如果存在解析解，画数值解 vs 解析解的差值
- **已知 benchmark 复现**: 与文献 / 教科书结果对比
- **定性合理性**: 轨迹是否光滑、对称性是否正确

### 结果图
附 1-3 张截图，放在项目目录下的 `assets/` 中。
